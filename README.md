# CFWarp

## Описание

Многофункциональный WARP-скрипт для VPS с установкой в один клик.  
Поддерживает VPS с **чистым IPv4** и **чистым IPv6**, а также основные Linux-системы.

---

## Установка

Используйте одну из команд ниже:

```bash
bash <(wget -qO- https://raw.githubusercontent.com/MyNicknme/warp/refs/heads/main/CFwarp.sh)

или

```bash
bash <(curl -Ls https://raw.githubusercontent.com/MyNicknme/warp/refs/heads/main/CFwarp.sh)

Важное замечание

Если после установки возникают такие проблемы, как:

потеря IP,

подвисание VPS,

ошибка загрузки при выполнении скрипта,

невозможность войти в меню скрипта,

сначала остановите WARP, а затем перезагрузите или переустановите его.
