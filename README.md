## Описание и важные замечания

Пояснения по использованию и важные замечания смотрите в видео по серии WARP и в журнале обновлений.

### 1. Многофункциональный WARP-скрипт в один клик

Поддерживает прямую установку на VPS с чистым **IPv4** и чистым **IPv6**.  
Поддерживаются основные Linux-системы.

```bash
bash <(wget -qO- https://raw.githubusercontent.com/MyNicknme/warp/refs/heads/main/CFwarp.sh)

или

bash <(curl -Ls https://raw.githubusercontent.com/MyNicknme/warp/refs/heads/main/CFwarp.sh)
Важно: если появляются такие проблемы, как:

потеря IP,

подвисание VPS,

ошибка загрузки при выполнении скрипта,

невозможность войти в меню скрипта,

сначала остановите WARP следующими командами, затем перезагрузите или переустановите WARP.

Остановить warp-go:

kill -15 $(pgrep warp-go)

Остановить wgcf:

systemctl stop wg-quick@wgcf
